# Customer Satisfaction Analysis

## 📌 Overview
This project is an introductory Natural Language Processing (NLP) analysis focused on evaluating customer satisfaction for an online women's clothing shop. By processing and analyzing raw text reviews, this project extracts actionable insights regarding customer sentiment across various product lines.

## 🎯 Objective
The primary task is to analyze textual customer reviews to determine which of the following clothing categories has the highest customer satisfaction:
* Tops
* Bottoms
* Jackets
* Dresses
* Intimate

## 🛠️ Tech Stack & Tools
* **Language:** Python
* **Environment:** Jupyter Notebook
* **Data Manipulation:** pandas
* **NLP & Text Processing:** NLTK (Natural Language Toolkit)

## 🧠 Key Learnings & Skills Demonstrated
* **Text Data Analysis:** Extracting meaningful patterns from unstructured text data.
* **Text Preprocessing:** Cleaning and preparing raw text for analysis using NLTK.
* **Sentiment Evaluation:** Applying NLP methodologies to classify and evaluate customer sentiment based on review content.

## 🚀 Getting Started

### Prerequisites
To run this notebook, you will need prior working knowledge of Python and pandas. Ensure you have the following installed on your local machine:
* Python 3.x
* Jupyter Notebook

### Installation & Setup

1. Clone the repository:
   git clone https://github.com/Raaaj2005/Customer-Satisfaction-Analysis.git

2. Navigate into the downloaded folder:
   cd Customer-Satisfaction-Analysis

3. Install the required libraries:
   pip install pandas nltk jupyter

4. Launch the Jupyter Notebook environment:
   jupyter notebook

5. Open the `Customer-Satisfaction-Analysis.ipynb` file in your browser and execute the cells sequentially.
